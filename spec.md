# Specification

## Summary
**Goal:** Add a curated set of property/lifestyle images inspired by the Nano Banana aesthetic to the landing page, replacing or supplementing existing placeholder imagery in the HeroSection, FeaturesSection, and AboutSection.

**Planned changes:**
- Replace or supplement the HeroSection background with a new warm-toned colonial Querétaro property image

- Replace or supplement the AboutSection image with a professional team photo
- Store all new images as static assets under `frontend/public/assets/generated`
- Update component references to point to the new image assets

**User-visible outcome:** The landing page displays a cohesive set of warm terracotta/earth-tone property and lifestyle images across the hero, features, and about sections, with no broken image links.
